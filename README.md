Simple CLI task-tracker.

You can create and change your tasks.

### Command list:
- **add** <description> - add new task
- **delete** <task_id> - delete task
- **update** <task_id> <description> - update description of task
- **mark** <task_id> <status> - change status of task
- **show** <filtered_by> - show your task filtered by status if <filtered_by> is given
